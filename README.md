# Aman-senpai Tap

## How do I install these formulae?

`brew install aman-senpai/tap/<formula>`

Or `brew tap aman-senpai/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "aman-senpai/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
